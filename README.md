# Label Station

Label Station prints DYMO fixture labels from a grandMA3 patch.

## Download

Go to **Releases** and download:

`LabelStation_Windows_Portable.zip`

Unzip it, then run:

`Start_LabelStation_Real_Print.bat`

Open:

`http://127.0.0.1:5000`

## Requirements

- Windows 10/11
- DYMO Connect installed
- DYMO printer working in DYMO Connect first
- grandMA3 onPC or console on the same network

## MA3 Plugin

Import or paste the plugin from:

`ma_plugins/LabelStation_PrintFromPatch_V4_8_ObjectList.lua`

Use the laptop IP shown on the Label Station status page.
